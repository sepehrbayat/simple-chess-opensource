# Chess Piece Images

این پوشه برای نگهداری تصاویر مهره‌های شطرنج استفاده می‌شود.

## نام‌گذاری فایل‌ها

برای استفاده از تصاویر واقعی، فایل‌های زیر را در این پوشه قرار دهید:

### مهره‌های سفید:
- `wp.png` - سرباز سفید (Pawn)
- `wn.png` - اسب سفید (Knight)
- `wb.png` - فیل سفید (Bishop)
- `wr.png` - رخ سفید (Rook)
- `wq.png` - وزیر سفید (Queen)
- `wk.png` - شاه سفید (King)

### مهره‌های سیاه:
- `bp.png` - سرباز سیاه (Pawn)
- `bn.png` - اسب سیاه (Knight)
- `bb.png` - فیل سیاه (Bishop)
- `br.png` - رخ سیاه (Rook)
- `bq.png` - وزیر سیاه (Queen)
- `bk.png` - شاه سیاه (King)

## منابع تصاویر رایگان

می‌توانید تصاویر مهره‌های شطرنج را از منابع زیر دانلود کنید:

1. **Chess.com Piece Images**: 
   - می‌توانید از تصاویر استاندارد شطرنج استفاده کنید
   - اندازه توصیه شده: 80x80 پیکسل یا بزرگتر

2. **OpenClipart**:
   - https://openclipart.org/
   - جستجو برای "chess pieces"

3. **Flaticon**:
   - https://www.flaticon.com/
   - جستجو برای "chess"

## نکات مهم

- فرمت فایل باید PNG باشد
- تصاویر به صورت خودکار به اندازه مناسب تغییر می‌کنند
- اگر تصاویر موجود نباشند، بازی از تصاویر ساده با نمادهای Unicode استفاده می‌کند
- پس‌زمینه تصاویر باید شفاف (transparent) باشد

## استفاده بدون تصاویر

اگر تصاویر را اضافه نکنید، بازی به صورت خودکار از تصاویر ساده با نمادهای Unicode استفاده می‌کند که به خوبی کار می‌کنند.
